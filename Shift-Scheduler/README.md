# Shift Scheduler

Starter MVC-style skeleton for a PHP-based shift scheduling system.

## Getting Started

1. Point your web server document root to `Shift-Scheduler/public`.
2. Update database credentials in `Shift-Scheduler/config/database.php`.
3. Visit `/` for the admin dashboard or `/login` for the login page.

## Structure

- `app/Controllers` - Request handlers
- `app/Models` - Data access layers
- `app/Views` - View templates
- `public/` - Web root assets and entry points
- `config/` - Application configuration

## Notes

This scaffold includes minimal routing, view rendering helpers, and placeholder API endpoints so you can begin implementing features immediately.
